# KI7MT Shack AI Hub

**The Model Context Protocol (MCP) ecosystem for the modern Amateur Radio operator.**
This repository is a collection of independent, specialized micro-servers designed to turn local AI agents into the ultimate station managers.

## The Toolset

* **[ADIF Validator](http://adif-mcp.com)** (`com.adif-mcp.validator`) - High-speed log analysis and 3.1.6 spec compliance.
* **WSPR Researcher** (`io.ki7mt.wspr`) - Query gigabytes of propagation archives on your local LAN.
* **QRZ Lookup** (`io.ki7mt.qrz`) - Real-time XML callbook integration.

## Quick Start

All tools are powered by `uv` for zero-configuration deployment.
```bash
# To run the ADIF validator:
uvx adif-mcp
```
